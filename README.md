# MILCO Child Theme
**Status: Work in Progress** 🚧

A custom WooCommerce child theme for MILCO — a Made-in-Lesotho 
marketplace connecting local artisans, co-operatives, and farmers 
with buyers nationwide.

## Built with
- WordPress + WooCommerce
- Storefront parent theme
- Custom PHP, CSS, JavaScript (A little page builder)

## Features implemented
- [x] Custom hero section with full-width cover
- [x] Trust bar (100% Lesotho-made, Verified producers, etc.)
- [x] WooCommerce product grid (4-column, branded cards)
- [x] "Made in Lesotho" badge on all products
- [x] LSL (Maloti) currency symbol
- [x] Custom rounded button system
- [x] Admin notifications bell (orders + low stock)
- [x] Low-stock dashboard widget
- [x] Custom footer with brand credit

## In progress

- [ ] Producer profile pages
- [ ] Mobile navigation drawer
